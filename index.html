GIT TIPS AND TRICKS

  • git status

  • Ця команда покаже, на якій гілці ви зараз працюєте. Якщо ви не на гілці main, ви можете переключитися на неї:

    git checkout main

  • Щоб зтягнути останні зміни з віддаленого репозиторію, використовуйте команду git pull. Ця команда оновить вашу локальну гілку до останнього стану віддаленої гілки main.

    git pull origin main

  • Якщо зміни у віддаленій гілці конфліктують з вашими локальними змінами, Git повідомить вас про це та надасть інструкції для вирішення конфліктів. Вам потрібно буде вручну вирішити ці конфлікти у файлах, а потім закомітити зміни.

  • Повний процес з прикладом:

    1.Перевірка поточної гілки:

    git status

    2.Перехід на гілку main:

    git checkout main

    3.Зтягування змін з віддаленої гілки main: git pull origin main
    • Якщо ви працюєте на іншій гілці, наприклад, на гілці розробки (feature-branch), і хочете зтягнути зміни з main до вашої поточної гілки, використовуйте команду merge:
  • 1.Перехід на вашу гілку розробки: git checkout feature-branch
  • 2.Зтягування змін з віддаленої гілки main: git pull origin main
  • 3.Злиття змін з main до вашої гілки: git merge main
